Import Secondary Sales

Upload secondary sales Data (stockist statements) for {{ division }} division — one month at a time

This creates one Stockist Statement (saved as Draft) per stockist for the selected month.
  • Stockists are matched strictly by name — the file's stockistcode is ignored. Unmatched names are skipped and reported.
  • Products are mapped by pcode only. Unknown codes are kept on the row as unmapped.
  • Rows with no movement (sales = 0 and free = 0) are skipped — previous-month closing carry-over, not current-month sales.
  • A free-only row (sales = 0, free > 0) is kept only when its statementdate is in the selected month; dated to a prior month it is skipped as carry-over.
  • Quantities are imported as-is (already final — no pack/box conversion). Values are computed as qty × rate.
  • The valuation rate is nrv (the net / final price rate) — equal to pts when no discount, lower when a discount was given. Falls back to pts if nrv is blank.
  • A stockist that already has a statement for the month is skipped — delete it first to re-import.
